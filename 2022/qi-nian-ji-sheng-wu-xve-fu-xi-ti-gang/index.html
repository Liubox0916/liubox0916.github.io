  2. 生物与环境的关系

    • 生物与环境的关系P16-P17：

      graph LR; B[生态因素] --> C(生物因素); B --> D(非生物因素); C --> E(影响生物生活的其他生物); D --> H(光); D --> I(温度); D --> J(水);

      环境中影响生物的生活和分布的因素叫做生态因素。生物在适应环境的同时，也影响和改变着环境例：骆驼刺的根系非常发达、骆驼排尿、出汗少，减少体内水分流失、生活在寒冷海域中的海豹皮下脂肪很厚、蚯蚓在土壤中活动，可以使土壤疏松......

  3. 生态系统相关内容；食物网和食物链等相关内容；生态系统能量的最初来源

    • 生态系统相关内容P19-P21：

      生态系统三点
      在一定的空间范围内，生物与环境所形成的统一的整体叫做生态系统。
      植物是生态系统中的生产者，动物不能自自己制造有机物，它们直接或间接的以植物为食，因此叫做消费者。细菌和真菌常被称为生态系统中的分解者。
      生产者、消费者和分解者之间是相互依存、相互制约的关系。

    • 食物网和食物链等相关内容P21-P22：

      食物网和食物链两点
      在生态系统中，不同生物之间由于吃与被吃的关系而形成的链状结构叫做食物链。
      在一个生态系统中，往往有很多条食物链，它们彼此交错连接，形成食物网。

    • 生态系统能量的最初来源：

      太阳能

  4. 多种多样的生态系统（地球之肺、地球之肾）；生态系统中各生物数量动态平衡

    • 多种多样的生态系统P28-P29：

      生态系统七点 称号 说明
      草原生态系统 / 分布在干旱地区，有水土保持和防风固沙等重要作用
      湿地生态系统 地球之肾 沼泽是典型的此系统，有净化水质、蓄洪抗旱作用
      海洋生态系统 / 制造的氧气占地球每年产生氧气总量的70%
      森林生态系统 地球之肺绿色水库 分布在较湿润地区，涵养水源，保持水土，防风固沙，调节气候，净化空气
      淡水生态系统 / 包括河流、湖泊、池塘生态系统，提供饮用、灌溉及工业用水资源调节气候
      农田生态系统 / 以农作物为主体，具有抵抗旱涝和病虫害能力差的特点，需要投入大量的人力物力
      城市生态系统 / 人类起着重要的支配作用，排放的污水、废气和固体废物较多，容易产生环境问题

    • 生态系统中各生物数量动态平衡P23：

      生态系统中各种生物的数量和所占的比例总是维持在相对稳定的状态，具有一定的自动调节能力，越复杂的生态系统自我调节能力越强。

  5. 显微镜使用的相关问题；制作并观察洋葱内表皮、人口腔上皮细胞临时装片

    • 显微镜使用的相关问题P37-P39:

      显微镜使用三点 详细步骤
      取镜和安放 将显微镜从镜箱中取出，另一只手握住镜壁，另一手另一只手托住镜座，把显微镜放在实验台距边缘大约七厘米处，安装好目镜和物镜。
      对光 转动转换器，使低倍物镜对准通光孔，物镜前端与载物台要保持两厘米左右距离，用一个较大的光圈对准通光孔，一只眼注视目镜内，转动反光镜，使反射光线经过通光孔物镜镜筒达到目镜，通过目镜看到明亮的圆形视野为宜。
      观察 把所要观察的玻片标本正面朝上，放在载物台上，用压片夹夹住拨片标本，要正对通光孔的中心转动粗准焦螺旋，使镜筒缓慢下降，直到物镜接近拨片标本为止。此时，眼睛一定要侧面看着物镜，一只眼向目镜内看。同时逆时针方向旋转粗准焦螺旋，使镜头缓缓上升，直到看清物象为止，再略微转动细准焦螺旋，使看到的物象更加清晰。

    • 制作并观察洋葱内表皮P42-P43:

      洋葱内表皮三点（擦滴盖染吸） 详细步骤
      准备 用洁净的纱布将将载玻片和盖玻片擦拭干净，将载玻片放在实验台上，用滴管在载玻片的中央滴一滴清水。
      制作临时装片 用镊子从洋葱鳞片叶内侧撕取一小块透明薄膜内表皮，把撕下的内表皮浸入载玻片上的水滴中，并用镊子将它展平。用镊子夹起盖玻片，使它的一边先接触载玻片上的水滴，然后缓慢的放下，盖在要观察的洋葱内表皮上，避免盖玻片下出现气泡
      染色 把一滴碘液滴在盖玻片的一侧，用吸水纸从盖玻片的另一侧吸引，使碘液浸润标本的全部

    • 人口腔上皮细胞临时装片P46-P47:

      人口腔上皮细胞六点（擦滴刮涂盖染吸）
      用洁净的纱布将载玻片盖玻片擦拭干净。
      在载玻片中央滴一滴生理盐水。
      用消毒牙签在自己肃静的口腔内侧壁上轻刮几下，把牙签附有碎屑的一端放在载玻片上的生理盐水中轻涂几下。
      用镊子夹起盖玻片，使它的一边先接触载玻片的水滴。然后将盖玻片缓缓放平，盖在水滴上，注意避免盖玻片下出现气泡。
      在盖玻片的一侧滴加几滴稀碘液。
      用吸水纸在盖玻片的另一侧吸引，使碘液浸润标本的全部。

  7. 染色体概念、组成，细胞分裂中染色体数目变化

    • 染色体概念P57：

      细胞中那些被碱性染料染成深色的物质称作染色体。

    • 染色体组成P57：

      染色体是由DNA和蛋白质两种物质组成的。DNA是遗传物质，因此可以说染色体是遗传物质的载体。

    • 细胞分裂中染色体数目变化P57：

      graph LR; Z[细胞]-->A(染色体); A --复制--> B(染色体); A --复制--> C(染色体); Z --> Y[新细胞]; Z --> X[新细胞]; B --> X; C --> Y;

    细胞在分裂时，染色体进行复制后均分成完全相同的两份，进入两个新细胞，复制时染色体翻倍，新细胞的染色体数目与旧细胞相同。

    • 细胞分化P57：

      在个体发育过程中，一个或一种细胞通过分裂产生的后代，在形态、结构和生理功能上发生差异性的变化，这个过程叫做细胞分化。

    • 人体组织P59-P60：

      每个细胞群都是由形态相似，结构、功能相同的细胞联合在一起形成的，这样的细胞群叫做组织。

      人体组织四点 组成细胞 功能
      上皮组织 上皮细胞 具有保护、分泌等功能
      肌肉组织 肌细胞 有收缩、舒张功能
      神经组织 神经细胞 能够感受刺激，传导神经冲动，有调节和控制作用
      肌肉组织 / 结缔组织的种类很多：骨组织、血液等都属于结缔组织，有保护、支持、连接、营养等作用

    • 花的主要结构P104：

    • 传粉过程P104：

      花药成熟后会自然裂开，散发出花粉。花粉从花药中散落而落到雌蕊柱头上的过程叫做传粉。植物传粉的方式一般有两种类型，自花传粉和异花传粉。异花传粉需要一定的媒介，这种媒介主要是蜂和虫。

    • 果实和种子的形成P106:

      受精完成后，花瓣、雌蕊以及柱头和花柱都完成了“历史使命”，纷纷凋落。子房发育成果实，子房壁发育成果皮，子房里面的胚珠发育成种子，胚珠里面的受精卵发育成胚。

    • 果穗缺粒的原因P106:

      传粉不足

  16. 绿叶在光下制造有机物实验

    • 绿叶在光下制造有机物实验P117:

      绿叶在光下制造有机物
      把盆栽的天竺葵放在黑暗处一昼夜。
      用黑纸片儿把叶片的一部分，从上下两面遮盖起来，然后移到阳光下照射。
      几小时后，摘下叶片，去掉遮光的纸片。
      把叶片放入盛有酒精的小烧杯中水域加热，使叶片含有的叶绿素溶解到酒精中，叶片变成黄白色。
      用清水漂洗叶片，再把叶片放到培养皿中，向叶片滴加碘液。
      稍停片刻，用清水冲掉碘液，观察叶片发生了什么变化。

  17. 光合作用概念；光合作用在农业生产中的应用；合理密植

    • 光合作用概念P124：

      绿色植物通过叶绿体利用光能把二氧化碳和水转化成储存能量的有机物并释放出氧气的过程。

      $[{\text{二}\text{氧}\text{化}\text{碳}+\text{水} \xrightarrow [{\text{叶}\text{绿}\text{体}}]{\text{光}\text{能}}\text{有}\text{机}\text{物}+\text{氧}\text{气}}]$

    • 光合作用在农业生产中的应用P125：

      合理密植

    • 合理密植P125：

      如果种植过稀,农作物没有充分利用单位面积上光照而造成浪费，如果种植过密，植株叶片互相遮挡，会影响植物的光合作用，影响产量。这两种都影响产量，应该合理密植。

  18. 呼吸作用概念；呼吸作用相关实验；呼吸作用在农业生产中的应用

    • 呼吸作用概念P129：

      细胞利用氧将有机物分解成二氧化碳和水，并且将储存在有机物中的能量释放出来，供给生命活动的需要，这个过程叫做呼吸作用。

      $[{\text{有}\text{机}\text{物}+\text{氧}\text{气} \xrightarrow {}\text{二}\text{氧}\text{化}\text{碳}+\text{水}+\text{能}\text{量}}]$

    • 呼吸作用相关实验P129：

      实验
      验证种子在萌发过程中是否产生二氧化碳的实验
      验证种子在萌发过程中是否消耗氧气的实验

    • 呼吸作用在农业生产中的应用P129：

      农田适时松土，遇到涝旱时及时排水是为了保证植物的根得到充分的氧气，保证呼吸作用正常进行。相反，有的措施是为了降低呼吸作用的强度，例如储存粮食时保持干燥和低温，储藏水果蔬菜时降低温度和氧浓度。
